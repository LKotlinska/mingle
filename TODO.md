# Networking Tool – WU + DD 2026

## Project Idea

A mobile-first networking tool that helps students and companies connect during the event at Visual Arena.

Users can register as either a student or a company, present themselves, and find relevant contacts for internships.

---

## MVP (First Version)

- [ ] Be mobile first
- [ ] Have a landing page
- [ ] Let the user choose:
  - [ ] Student
  - [ ] Company
- [ ] Show a form based on the selection
- [ ] Save user data
- [ ] Display registered profiles
- [ ] Be deployed
